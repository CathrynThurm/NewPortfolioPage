Portfolio

Blur License Plates

This program uses computer vision to detect and blur Russian license plates using Python and haar feature-based cascade classifiers.

This program is for anyone who would like to add a layer of privacy to their photos of cars and was built for practicing computer vision using Open CV.

Check out the Github repository.

Cancer Prediction

This program uses machine learning to predict with %97 accuracy whether a tumor is malignant or benign. Taking over 500 data sets this program makes predictions based off of things like smoothness and concavity.

This program can be for medical professionals and was built for practicing machine learning using Scikit-Learn.

Check out the Github repository.
